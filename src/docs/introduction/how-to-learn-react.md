# React是什么？
React 是一个用于构建用户界面的 JavaScript 库。它由 Facebook 开发，已经成为了构建现代 Web 应用程序的主要工具之一。React 的主要特点是组件化和声明式编程，它可以让你更轻松地构建复杂的用户界面，并提高代码的可重用性和可维护性。
